Sailfish OS

Sailfish OS is a mobile operating system combining the Linux kernel for a particular hardware platform use, the open-source Mer core middleware, a proprietary UI contributed by Jolla, and other third-party components.
(Wikipedia)

Now you can install this OS to your Samsung Galaxy Tab 2 3G. Just follow the instructions below.

Installing custom ROMs may break your device. Use it on your own risk. I’m not responsible for any problems.

Getting started

Prerequisites

  • Samsung Galaxy Tab 2 3G (p5100 or p3100)
  • Installed TWRP recovery
  • Downloaded CyanogenMod 12.1 and Sailfish OS
  • The basic linux & android knowledge
  • Patience

Downloads

  • TWRP from Android File Host
  • CyanogenMod 12.1 from Android File Host
  • Sailfish OS from Google Drive
  • Cute kitten gif from Gfycat

Install

  • Copy downloaded CyanogenMod and Sailfish OS archives to root of sdcard
  • Boot to recovery
  • Go to Wipe -> Advanced and choose:
    • Dalvik Cache
    • System
    • Data
    • Cache
  • Flash CyanogenMod and then Sailfish OS zips
  • Reboot

Install process will take some time. Be patience.
